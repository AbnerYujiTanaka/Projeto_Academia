# Imagens do Projeto

## Imagem de Login (Wallpaper)

Para adicionar a imagem de wallpaper na página de login:

1. Coloque sua imagem nesta pasta (`assets/images/`)
2. Renomeie o arquivo para: `login-wallpaper.jpg` (ou `.png` se for PNG)
3. Formatos suportados: JPG, PNG, WebP

### Especificações recomendadas:
- **Resolução**: 1920x1080 ou superior
- **Formato**: JPG ou PNG
- **Tamanho**: Otimize para web (recomendado: menos de 2MB)

A imagem será exibida como wallpaper de fundo na página de login com um overlay escuro para garantir a legibilidade do formulário.
